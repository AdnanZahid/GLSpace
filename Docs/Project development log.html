GLSpace

A framework for building 3D websites.

URL: http://adnan-zahid.eu5.org/GLSpace

GLSpace is an open source project I am doing with Liquid Galaxy for Google Summer of Code 2014. My name is Adnan Zahid and my mentor is Ben Goldstein (President of Endpoint, NY).

Description of project:

This project is built using Three.js. It currently comprises of a stage with 4 walls. These 4 walls are used for displaying 4 webpages. The user can navigate around the stage to browse the internet in 3D.

23rd July:

Finally got the code working:

function captureScreen(callback) {

    var video_constraints = {

        mandatory: {

            chromeMediaSource: 'screen'

        },

        optional: []

    };

    navigator.webkitGetUserMedia({

        audio: false,

        video: video_constraints

    }, function (stream) {

        callback(stream);

    }, function (error) {

        console.error(error);

    });

}

Now trying to port it in my ThreeJS application.
